# CSAPP Data Lab Instructions

This project is CSAPP Data Lab.

When editing bits.c:
- Only modify student solution functions.
- Do not change function signatures.
- Do not modify tests or build files unless explicitly requested.
- Follow each function's Legal ops and Max ops.
- Do not use control flow unless the function rules allow it.
- Do not use function calls.
- Do not use macros.
- Do not use constants larger than 0xFF unless allowed by the problem.
- After edits, run:
  - make
  - ./dlc bits.c
  - ./btest
- Explain each solution in Chinese with:
  - problem translation
  - key idea
  - bit-level reasoning
  - final expression
