# CSAPP Labs

这个仓库用于记录我学习《Computer Systems: A Programmer's Perspective》（CS:APP）过程中完成的实验代码、笔记和调试记录。

CSAPP 是一门面向程序员的计算机系统课程，内容涵盖数据表示、机器级程序、程序优化、存储器层次结构、链接、异常控制流、虚拟内存、系统级 I/O、网络编程和并发等内容。

---

## 仓库目的

本仓库主要用于：

1. 保存 CSAPP 各个 Lab 的代码实现
2. 记录实验过程中的问题和解决方法
3. 加深对计算机系统底层机制的理解
4. 练习 Linux、GCC、GDB、Makefile 等工具的使用
5. 为后续学习操作系统、网络编程、C/C++ 后端开发打基础
